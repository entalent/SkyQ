# SkyQ

## Introduction

> An IM software that runs on linux, using GTK+ 2.0</br>
> This project used cJSON library to parse and generate JSON strings, and SQLite3 to store user data and chat record. 

## Functions

- Peer-to-peer chat
- Group chat
- Chat record storage
- Offline message
